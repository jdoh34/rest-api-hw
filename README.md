# REST API 

A simple REST API built with Express.js to manage a small in-memory homework list. This project supports creating, reading, and deleting homework items.

## Features

- View all homework assignments
- View a single homework assignment by name
- Add a new homework assignment
- Delete a homework assignment
- Return the total homework count with a `HEAD` request
- Uses in-memory storage for simplicity

## Tech Stack

- Node.js
- Express.js
- Nodemon
